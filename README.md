# CODSOFT Task 4 - Recommendation System

## 📌 Project Overview
This project is a simple Recommendation System developed using Python. It recommends movies based on the user's preferred genre using predefined categories. The project demonstrates the basic concept of recommendation systems and user preference-based suggestions.

## ✨ Features
- Recommends movies based on selected genre
- Interactive command-line interface
- Supports multiple movie genres
- Easy to use and modify
- Beginner-friendly implementation

## 🛠️ Technologies Used
- Python 3

## ▶️ How to Run
1. Clone the repository.
2. Open the project folder.
3. Run the following command:

```bash
python recommendation_system.py
```
